# Android-Compose-MVVM-Clean-Weather-App

## Mohammed Badsha Alamgir

Android Weather app using Jetpack Compose, MVVM Clean Architecture, Flow, Retrofit

## How to Run

1. Open Project in android studio

2. Minimum SDK `26`
3. Compile SDK `32`
4. Use `Java 11`
5. Sync `build.gradle` file
6. Run project
